# GitHub Copilot Labs 总览

| Lab | 名称 | 内容概述 |

| 07 | [CodingAgent](./07.CodingAgent.md) | 用 GitHub Coding Agent 自动根据 Issue 实现代码。 |
| 08 | [ReviewAgent](./08.ReviewAgent.md) | 用 GitHub Review Agent 自动审查 PR。 |
| 09 | [CopilotCLI](./09.CopilotCLI.md) | 使用 Copilot CLI 进行命令与代码审查。 |
| 10 | [CopilotCLIIntegration](./10.CopilotCLIIntegration.md) | 在 GitHub Actions 中集成 CLI + Azure OpenAI。 |
| 11 | [CodeSpace](./11.CodeSpace.md) | 在 GitHub Codespace 中使用 GitHub Copilot和Copilot CLI。 |
| 12 | [CopilotSpace](./12.CopilotSpace.md) | 通过 Copilot Space 管理知识库并在 VS Code 中调用 MCP 服务。 |
