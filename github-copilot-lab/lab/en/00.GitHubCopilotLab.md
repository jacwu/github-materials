# GitHub Copilot Labs Overview

| Lab | Name | Summary |
|-----|------|---------|
| 01 | [CopilotAgent](./01.CopilotAgent.md) | Generate a PRD in VS Code and bootstrap a travel site project. |
| 02 | [CustomizeInstructions](./02.CustomizeInstructions.md) | Create global and language-specific instruction files to standardize coding conventions. |
| 03 | [PromptAndChatMode](./03.PromptAndChatMode.md) | Build reusable prompt files and custom chat modes. |
| 04 | [MCPServer](./04.MCPServer.md) | Configure and use an MCP server in VS Code. |
| 05 | [CopilotChat](./05.CopilotChat.md) | Use Copilot Chat on GitHub.com for Q&A. |
| 06 | [CodingAgent](./06.CodingAgent.md) | Use GitHub Coding Agent to automatically implement code from issues. |
| 07 | [ReviewAgent](./07.ReviewAgent.md) | Use GitHub Review Agent to automatically review pull requests. |
| 08 | [CopilotCLI](./08.CopilotCLI.md) | Use Copilot CLI for command assistance and code reviews. |
| 09 | [CopilotCLIIntegration](./09.CopilotCLIIntegration.md) | Integrate the CLI and Azure OpenAI in GitHub Actions. |
| 10 | [CodeSpace](./10.CodeSpace.md) | Use GitHub Copilot and Copilot CLI inside GitHub Codespaces. |
| 11 | [CopilotSpaces](./11.CopilotSpaces.md) | Manage knowledge bases with Copilot Spaces and call MCP services in VS Code. |
