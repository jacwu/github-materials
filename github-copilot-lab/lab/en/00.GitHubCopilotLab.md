# GitHub Copilot Labs Overview

| Lab | Name | Summary |
|-----|------|---------|
| 01 | [CopilotAgent](./01.CopilotAgent.md) | Generate a PRD in VS Code and bootstrap a travel site project. |
| 02 | [CustomizeInstructions](./02.CustomizeInstructions.md) | Create global and language-specific instruction files to standardize coding conventions. |
| 03 | [SubAgent](./03.SubAgent.md) | Use SubAgent to handle complex tasks. |
| 04 | [PromptFile](./04.PromptFile.md) | Build reusable prompt files. |
| 05 | [MCPServer](./05.MCPServer.md) | Configure and use an MCP server in VS Code. |
| 06 | [CopilotChat](./06.CopilotChat.md) | Use Copilot Chat on GitHub.com for Q&A. |
| 07 | [CodingAgent](./07.CodingAgent.md) | Use GitHub Coding Agent to automatically implement code from issues. |
| 08 | [ReviewAgent](./08.ReviewAgent.md) | Use GitHub Review Agent to automatically review pull requests. |
| 09 | [CopilotCLI](./09.CopilotCLI.md) | Use Copilot CLI for command assistance and code reviews. |
| 10 | [CopilotCLIIntegration](./10.CopilotCLIIntegration.md) | Integrate the CLI and Azure OpenAI in GitHub Actions. |
| 11 | [CodeSpace](./11.CodeSpace.md) | Use GitHub Copilot and Copilot CLI inside GitHub Codespaces. |
| 12 | [CopilotSpace](./12.CopilotSpace.md) | Manage knowledge base with Copilot Space and call MCP server in VS Code. |
