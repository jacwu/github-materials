## Lab Environment

### Software Requirements
- **Node.js**: >= 22.0.0
- **npm**: >= 10.0.0
- **VS Code**: latest version
- **GitHub Copilot**: signed in

---

## Lab Steps

#### 1.1 Goal
Use Copilot CLI to review an existing code file.

#### 1.2 Procedure

1. **Install Copilot CLI**
   Run:
   ```bash
   npm install -g @github/copilot
   ```

2. **Launch Copilot CLI**
   In the terminal, navigate to the project folder and type `copilot` to sign in.
   ![launch copilot](../images/091.png)

3. **Explore commands**
   Press `/` to view all supported commands.
   ![check command](../images/092.png)

4. **Run a code review**
   Enter `review @sort.ts` so Copilot reviews `sort.ts` (replace the path as needed).
   ![review](../images/093.png)

#### 1.3 Validation
- Copilot CLI runs successfully and provides a review for `sort.ts`.
