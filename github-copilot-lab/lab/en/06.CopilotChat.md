## GitHub Copilot Lab

### What Is Copilot Chat on the Web?

The web version of Copilot Chat lives on GitHub.com and is accessible from repositories, pull requests, issues, discussions, and the knowledge base. The responses are tailored to the context you have open when you launch the chat.

### How We'll Use It in This Lab
We'll open the repository on GitHub.com and use Copilot Chat to ask questions.

---

## Lab Environment

### Software Requirements
- **Node.js**: >= 22.0.0
- **npm**: >= 10.0.0
- **VS Code**: latest version
- **GitHub Copilot**: signed in

---

## Lab Steps

#### 1.1 Goal
Use Copilot Chat on the web.

#### 1.2 Procedure

1. **Open the repository**
   Sign in to GitHub.com and navigate to the repository you created earlier.

2. **Launch Copilot Chat**
   Click **Chat with Copilot** to open the Copilot Chat interface.
   ![Copilot reference example](../images/061.png)

3. **List issues**
   In Copilot Chat run:
   ```
   list all the issues and summarize
   ```
   ![Copilot reference example](../images/062.png)

4. **Create an issue**
   Run:
   ```
   /create-issue Add a "My Orders" page where users can review their past orders
   ```
   ![Copilot reference example](../images/063.png)

5. **Confirm the issue**
   Review the proposed issue details and click **Create** to submit it.
   ![Copilot reference example](../images/064.png)

#### 1.3 Validation
- Issues are successfully listed and the new issue is created.
